# Welcome ⚡

I'm Richard Wollyce, a software engineer focused on systems, security, and execution.

I work across TypeScript, JavaScript, Python, Bash, PostgreSQL, Linux, and whatever the problem requires.

I think in systems before I build them.

I like to visualize the structure, understand the moving parts, project outcomes, and only then execute.
Most problems have more than one possible solution. I prefer to map the options, explain the tradeoffs, and build with intent.

## About

- I build with a systems mindset. Not just features, but structure, constraints, behavior, and long-term maintainability.
- At MG Laser, I build and maintain an internal ERP that supports daily operations across multiple teams and 200+ orders per day.
- I replaced fragile manual workflows with structured flows and validation, reducing data entry errors by about 95%.
- I work close to production when needed. Deployment, monitoring, VPS, access rules, and incident recovery.
- My current direction is cybersecurity, infrastructure, and deeper technical rigor.
